# Glossary

- **ULA**: Unique Local Address space for IPv6 private internal routing.
- **/48**: Prefix length used here as the organization-level IPv6 allocation.
- **/56**: Site-level prefix allocation carved from the `/48`.
- **/64**: Standard subnet size for IPv6 LAN segments.
- **EVPN**: Ethernet VPN control plane, often used with VXLAN overlays.
- **VXLAN**: Overlay encapsulation that enables logical Layer 2/3 segments over IP fabrics.
- **BGP**: Border Gateway Protocol for dynamic route exchange.
- **Anycast**: Same IP advertised from multiple locations for nearest-path service access.
- **RTO**: Recovery Time Objective, the target time to restore service.
- **RPO**: Recovery Point Objective, the maximum acceptable data loss window.
- **SDN**: Software-Defined Networking, policy and topology control through software abstraction.
